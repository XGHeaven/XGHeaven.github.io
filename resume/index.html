独立项目

CNode 第三方 Web

使用 Angular-Material 作为主要开发框架。配合使用 gulp，browserify，browser-sync 等工具。实现了页面的资源的自动化编译压缩，配合 browser-sync 可以实现 save-reload 功能。

口袋杭电微信墙

用 Node.js 开发，通过简易的 Websocket 协议建立客户端和服务器之间的长连接，借助 Mongodb 存储引擎，存储下所有的弹幕并向所有用户推送。配合上自制的弹幕流量控制，可以保证弹幕出现的频率适中。完全前后端分离开发，接口符合 RESTful 规范。可以动态停止开启微信墙，不需要重启服务器。

红色家园云 U 盘

主要是使用 php 开发类似于匿名网盘的项目。主要难点在于防止恶意上传和存储空间的清理。使用了 MD5 码对上传文件进行判断重复。每天都会启动分析程序，对这一天清理了多少内容，上传了多少内容进行分析统计。通过 Web 呈现出来。

env-deploy

Node.js 包，主要用于开发环境下可以自动将 .env 文件中配置的环境变量加载到 process.env 中。方便开发和部署。

think-css-preprocess

ThinkJS 的自动对静态文件进行编译，主要支持 Stylus,Sass,Less CSS 文件。
