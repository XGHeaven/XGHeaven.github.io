Experience

  • 2015.07.22-2015.10.24 口袋高校微信弹幕墙


    • 这次我负责全栈开发，主要面向百人以上的活动弹幕的承接，提供自主开启关闭微信墙，自主过滤，自动通过等功能。

    • 前端部分：手机端和展示端主要使用了 Angular 框架，因为那个时候 React 和 Vue 还没有流行起来。采用前后端分离，接口符合 RESTful 规范。对 Angular 框架有比较多的了解，清楚其脏值检测的原理，了解页面刷新的时机和标签的使用。配合 WebSocket 协议，实现事实弹幕刷新和发送。

    • 后端部分：采用 express 构建的 RESTful 接口以及 socket.io 提供的 WebSocket 的支持。可以很好的实现实时应用的构建。存储配合 mongodb。



  • 2015.09.15 - 至今 CNode PC 网页客户端，Material Design 风格



    • 采用 Material Design 风格设计的 CNode.js 中文社区

    • 主要使用 Angular-Material 框架实现 Material Design 的效果。这次也主要使用了 browserify 对代码进行打包，之所以没有使用 Webpack，因为觉得不需要功能那么强大的打包工具。创新使用了模态窗口查看文章的方式，免去了前进后退的时间，减少网络请求。



  • 2015.10.06 - 至今 杭州电子科技大学红色家园“在杭电”微信公众号后台



    • 主要使用 express 和 node-wechat 框架进行开发，使用 es6/es7 语法以及 babel 对其进行转码，适用于更低的 node 版本。使用 mongodb 作为存储引擎，使用 redis 作为缓存引擎，使用 docker 作为分发方式。较为熟练的使用 mocha/istanbul/sinon mock 对代码进行 BDD 测试。

    • 通过这个也了解到了大量的与 Node.js 相关的技术实现设计等。



  • nodeclub CNode.js 中文贡献了点击图片放大的 feature



    • 在 CNode 也待过挺长的一段时间，也做过很多有用的回答，也感觉到了 Node 社区的活跃，心中很是欣慰。不过 CNode 的站长似乎不是很想网站改版，这有点让人僵硬。
